## The Problem

Peaksy hikers need mobile-native access to profiles, events, and social features — not just a responsive web wrapper.

## What I Built

Cross-platform mobile features for Peaksy at Symph, mirroring core web social and discovery flows in a React Native app.

## How It Works

React Native UI components connect to the same NestJS API layer as the web app. Shared TypeScript types and API contracts keep web and mobile behavior consistent across the monorepo.

## Outcome / Results

Extended Peaksy's production footprint to mobile with reusable API-connected patterns validated on both platforms.
