## Overview

This project is loaded from an MDX file in `content/projects`. Edit the frontmatter and body to add your own case studies.

## Stack

- **Framework:** Next.js 15 App Router
- **Styling:** Tailwind CSS v4 + shadcn/ui
- **Content:** File-based MDX with gray-matter

## Next steps

Replace this file with your own projects, or add more `.mdx` files to the same folder. Each file becomes a route at `/projects/[slug]`.
