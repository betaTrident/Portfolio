## The Problem

Outdoor enthusiasts lack a dedicated social platform for discovering hikes, saving events, and building a hiker-focused social graph.

## What I Built

Full-stack Peaksy web features at Symph — profile and social graph functionality, search and discovery flows, saved-event experiences, and review features with API integrations.

## How It Works

Next.js frontend with reusable UI components connected to NestJS APIs. Supabase/PostgreSQL stores user profiles, events, and social relationships. API-connected workflows support scalable, responsive discovery and engagement.

## Outcome / Results

Shipped production-ready features in a Git-based monorepo. Delivered cross-cutting social and discovery modules used by real hikers on the platform.
